# Prerequisites
## 1.Domain Knowledge
### Bearing
[Envelope Analysis in MATLAB](https://www.kau-sdol.com/matlab-code)

    介绍了传统的故障诊断方法，为了提取故障特征频率，采用自回归+带通滤波+希尔伯特变换的pipline

[轴承参数计算app](http://www.jp.nsk.com/app02/BearingGuide/m/html/en/BearingSearch.html)
    
    输入型号计算剩余寿命，轴承故障频率等
